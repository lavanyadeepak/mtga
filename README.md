# Voice of [Your City] 🏙️
### Make [Your City] Great Again

A open-source, whitelabel civic complaint portal built for 
Indian municipal corporations.

## What makes this different from every other civic portal:
- Built-in corruption reporting (not an afterthought)
- Social-shareable complaint cards with IP stamping
- Bilingual out of the box (English + regional language)
- Ward map integrated
- #MTGA ready 😄
